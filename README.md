# 机械图纸智能气泡标注PDF编辑器

专为机械制造、质检行业开发的PDF图纸标注工具，支持智能识别形位公差、尺寸标注并自动生成气泡序号，同时兼容传统PDF编辑功能。

## 功能特性

### 智能识别
- ✅ 形位公差识别：支持14项GD&T几何公差符号与文字描述
- ✅ 线性尺寸识别：直径、半径、角度、对称公差、范围尺寸
- ✅ 技术注解识别：技术要求、热处理、粗糙度、材料说明

### 气泡标注
- ✅ 一键智能自动标注，按从上到下自动排序编号
- ✅ 连续标注模式：点击即可递增添加气泡
- ✅ 自定义气泡样式：大小、颜色、字号、边框
- ✅ 批量重新编号
- ✅ 导出标注清单

### 基础PDF功能
- ✅ 打开、保存、另存为PDF
- ✅ 多页面导航浏览
- ✅ 高清渲染显示
- ✅ 直线、文本、圆形注释

## 安装运行

### 环境要求
- Python 3.9 ~ 3.12
- Windows / Linux / macOS

### 快速启动
1. 解压所有文件到同一文件夹
2. Windows双击 `run.bat`，Linux/Mac执行 `bash run.sh`
3. 首次运行会自动安装依赖

### 手动安装
```bash
pip install -r requirements.txt
python main.py
